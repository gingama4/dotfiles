# dotfiles

[![Test](https://github.com/gingama4/dotfiles/actions/workflows/test.yml/badge.svg)](https://github.com/gingama4/dotfiles/actions/workflows/test.yml)

## Supported OS

- ArchLinux
- Ubuntu

## Install

1. Download

```bash
git clone https://github.com/gingama4/dotfiles.git
```

2. Install
```bash
./setup.sh
```

## Components

* bash
* vim
* tmux
* git
